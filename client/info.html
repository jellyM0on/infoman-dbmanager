Pasig ScholarDB

About

This project was made in partial fullfillment for the course COMP 010 - Information Management, aiming to develop a database system to store scholarship applications and provide a centralized storage of information for efficient application submission, review, and selection. The design was based on the Pasig City School Year 2023-2024 Scholarship 2nd Semester Renewal Form. It stores the applicantâ€™s personal information, family information, and educational background.

Implementation

This project is a web application using an ExpressJS server as the backend, MySQL for the database, and icon assets from Google Fonts.
